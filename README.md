## Overview
- **Goal:** The ability to use my DT770 Pros wirelessly
- **Why:** Cables are messssyyyyy
- **Scope:** Power management and input toggling
- **Out of scope:** Single interaction source toggling (including powering down the bluetooth module)

## Highlights
- Toggle source control to isolate inputs
- USB-Rechargable
- Cheap parts list

## Requirements
- **Parts / Materials:** [bom.md](bom.md)
- **Tools:**
  - Soldering Iron + Tin (the lead-free kind naturally)
  - Heat Shrink Tubing / Electrical Tape
  - Single core wire
  - Wire sheath (optional)
  - Drill with varied drillbits
  - Hot glue / 2-part resin / malleable adhesive
- **Skills assumed:**
  - Basic soldering
